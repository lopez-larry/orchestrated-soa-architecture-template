# Orchestrated SOA Architecture Template

This repository represents a service-oriented architecture with a central orchestrator coordinating calls across services to complete business workflows.

## Folder Structure

orchestrator/
  workflows/  - workflow definitions and orchestration logic

services/
  service-a/  - example service
  service-b/  - example service

shared/
  schema/     - shared message or contract definitions

tests/        - tests for workflows and services

docs/
  architecture.md
  decisions.md
